# Nike Sales Analysis

## 📊 Project Overview
Analyzed 2500+ uncleaned Nike sales transactions using SQL & Python to identify revenue drivers and loss-making products.

## 🛠️ Tools Used
- SQL, Python, Pandas, Matplotlib, Jupyter Notebook

## 🔍 Key Insights
1. Found 6 unique cities after cleaning `bengaluru`, `Bangalore`, `hyd` duplicates
2. Online channel contributed 60% of total revenue vs Retail
3. 12% orders were returns identified by negative Units_Sold
4. SuperRep Go had highest loss rate among all products
5. Identified ₹50K potential loss from negative profit orders

## 📁 Files
- `sales_analysis.ipynb` - Complete EDA with Pandas
- `queries.sql` - Business questions solved using SQL
- `cleaned_data.csv` - Final cleaned dataset
