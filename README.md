# Nike Sales Analysis

## 📊 Project Overview
Analyzed 2500+ uncleaned Nike sales transactions to identify revenue drivers and loss-making products.

## 🛠️ Tools Used
- SQL, Python, Pandas, Matplotlib

## 🔍 Key Insights
1. Found 6 unique cities after cleaning `bengaluru, Bangalore, hyd` duplicates
2. Online channel contributed 60% of total revenue
3. 12% orders were returns with negative Units_Sold
4. SuperRep Go had highest loss rate
